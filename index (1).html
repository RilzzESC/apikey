🌐 Cara Kerja Integrasi

Data lisensi tersimpan di GitHub Gist (file JSON) menggunakan GitHub API. Validasi bisa dilakukan siapapun dengan membaca Gist secara publik — tanpa token. Operasi tulis (generate, revoke) hanya bisa dilakukan dengan token admin di halaman ini.

💡 Gist yang digunakan bisa public (siapapun bisa baca via URL) atau secret (hanya yang punya link). Keduanya bisa dibaca via API tanpa auth.
🔗 Endpoints (via Gist Raw)
GET
https://gist.githubusercontent.com/{user}/{gist_id}/raw/licenses.json
Baca semua data lisensi (JSON). Bisa diakses tanpa token. Cocok untuk client-side validation.
PATCH
https://api.github.com/gists/{gist_id}
Update data lisensi. Butuh GitHub token. Hanya dari admin panel ini.
💻 Integrasi di Aplikasi Anda

Validasi key dari aplikasi lain (JavaScript):

// Tidak perlu token! Gist raw bisa dibaca publik
const GIST_RAW = 'https://gist.githubusercontent.com/USER/GIST_ID/raw/licenses.json';

async function validateLicense(key) {
  // Fetch langsung dari Gist (no-cors tidak perlu, Gist support CORS)
  const res  = await fetch(GIST_RAW + '?t=' + Date.now()); // cache bust
  const data = await res.json();
  const entry = data.keys.find(k => k.key === key);

  if (!entry) return { valid: false, error: 'Key not found' };
  if (entry.status === 'revoked') return { valid: false, error: 'Revoked' };

  const today = new Date().toISOString().split('T')[0];
  if (entry.expires_at && entry.expires_at < today)
    return { valid: false, error: 'Expired', expired_at: entry.expires_at };

  const daysLeft = entry.expires_at
    ? Math.ceil((new Date(entry.expires_at) - new Date(today)) / 86400000)
    : null;

  return { valid: true, owner: entry.name, product: entry.product,
    expires_at: entry.expires_at, days_remaining: daysLeft };
}

// Contoh penggunaan:
const result = await validateLicense('XXXX-XXXX-XXXX-XXXX');
console.log(result);

Python:

import requests, datetime

GIST_RAW = 'https://gist.githubusercontent.com/USER/GIST_ID/raw/licenses.json'

def validate_license(key):
    data  = requests.get(GIST_RAW, params={'t': datetime.datetime.now().timestamp()}).json()
    entry = next((k for k in data['keys'] if k['key'] == key), None)
    if not entry: return {'valid': False, 'error': 'Not found'}
    if entry['status'] == 'revoked': return {'valid': False, 'error': 'Revoked'}
    today = datetime.date.today().isoformat()
    if entry.get('expires_at') and entry['expires_at'] < today:
        return {'valid': False, 'error': 'Expired'}
    return {'valid': True, 'owner': entry['name'], 'product': entry['product']}

print(validate_license('XXXX-XXXX-XXXX-XXXX'))
